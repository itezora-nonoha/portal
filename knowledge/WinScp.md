### 選択したファイルの内容をクリップボードにコピー
```
cat !
```
- [x] 結果をクリップボードにコピー

### 現在のディレクトリでファイル名検索
```
find !/ -name "!?検索パターン:?!"
```
- [x] 結果をクリップボードにコピー

### 選択したファイルをSJISに変換
```
iconv -f utf-8 -t cp932 ! -o `echo ! | `sed 's/.csv/_sjis.csv/g`
```
### 現在のディレクトリ内のファイル内容に対し、文字列検索（見つかったファイル内の行番号も出力）
```
grep -n -r !?検索パターン?! !
```
